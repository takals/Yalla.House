Gaps & Aspirational Features

What's not yet built or needs significant improvement. Ordered by priority.

Blocked

Stripe Products & Pricing

Plans page built, but Stripe products can't be created until company registration completes. Checkout flow is wired but has no real price IDs.

Reapit API (Portal Sync)

Portal badges are visual-only. Real Rightmove/Zoopla sync requires Reapit OAuth application — blocked on Sprint Zero Day 1.

Next Priority

Owner Settings (Notifications)

Currently a stub. Needs: notification preferences, email frequency, push notification opt-in, connected services management.

Dedicated Owner Email Addresses

{ref}@mail.yalla.house via SendGrid/SES inbound parse. Discussed in strategy but not built.

Real-time Messaging

Currently toast-only notifications. Needs: in-app messaging between owners and agents, Supabase Realtime channels.

Version 2 Aspirations

AI Property Valuation

Auto-generate comparable sales data using Land Registry API + AI analysis. Currently manual market data.

Virtual Viewings

3D property tours integration. Currently viewings are physical-only.

Agent Marketplace

Public agent profiles with ratings, specializations, and direct booking. Currently agents are matched behind the scenes.

Token Economics

Platform credits for services, referral rewards, agent performance bonuses. Strategy documented but not implemented.

Mobile App (PWA)

Progressive Web App shell for push notifications and offline listing drafts.
