## **WF-O2 Wizard — Questions Table (Step 1–5 + Publish Plan)**

| **Step** | **Step Name** | **What we ask (questions / fields)** | **Required?** | **Est. time** | **Advice (keep it smooth)** |
| --- | --- | --- | --- | --- | --- |
| 1 | Basics | **What’s the address?** (Address) • **What type is it?** (House/Flat/Studio/Other) • **Is this for sale or rent?** (Sell/Rent) | Address + Type + Sell/Rent = **Required** | 1–2 min | Create the listing record in **Draft immediately** after this step. Keep it very short; don’t ask anything commercial here. |
| 2 | Price (Listing data) | **What price do you want to list at?** (Numeric) • **How should the price be presented?** (Offers over / Negotiable / Shared Ownership) • *(Shared Ownership only)* **What % share?** + optional full value + optional rent estimate | Asking price + label = **Required** (SO extras conditional) | 1–2 min | This is **public listing price only**. Do **not** mix in NoAgent plan pricing. Keep currency auto-set by rollout country (read-only in Phase 1). |
| 3 | Details | **Bedrooms? Bathrooms?** • **Size?** (sqm/ft² + unit) • **Short description?** • *(Optional)* key features chips | Beds/Baths/Size/Description = **Required** | 2–4 min | Keep description prompts simple: “What would a buyer need to know in 3–5 sentences?” Avoid long form. Features chips optional. |
| 4 | Media & Documents | **Upload photos** (min 5) • *(Optional)* floorplan • *(Optional/country)* EPC • *(Optional)* other PDF docs | Photos (min 5) = **Required** | 3–8 min | This is the friction step. Make upload forgiving: drag/drop, progress, reorder later. Allow “Skip for now” for optional docs without blocking. |
| 5 | Finish & Preview | **Is everything complete?** (Checklist review) • Actions: **Preview Listing** (primary) / Save as Draft | Checklist + Preview CTA = **Required screen** | 1–2 min | Do **not** ask for package pricing here. The goal is confidence and preview. Always give “Save as draft.” |
| +1 | Publish Plan (after Preview) | **How do you want to publish?** (Self-service / Portal exposure / Premium) • *(If paid)* checkout + confirmation | Only required to go **Live** | 1–3 min | Trigger from **WF-O5 preview banner** or “Publish” button. If owner exits → remains Draft. Payment only when it costs you money (portals/services). |
