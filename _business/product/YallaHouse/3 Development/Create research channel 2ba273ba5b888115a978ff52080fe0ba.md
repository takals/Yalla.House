# 2) “How do you do sales progression and vendor updates today?”

**Likely agent answers**

- 
    
    “A spreadsheet + calls + WhatsApp. Every vendor asks ‘what’s happening?’”
    
- 
    
    “We send a weekly email—counts of viewings/offers, chain notes.”
    
- 
    
    “It’s inconsistent; if someone’s off, no one sees their notes.”
    

**Yalla.House response**

- 
    
    **Sales Progression pipeline**: Offer accepted → Searches → Enquiries → Survey → Mortgage offer → Exchange → Complete.
    
- 
    
    **Auto-nudges** to assigned negotiator when a step is overdue.
    
- 
    
    **Shared notes** per sale (no more ‘in Stacey’s phone’).
    
- 
    
    **Vendor Report generator**: last week’s activity, next steps, ETA—PDF or secure link.
    

# 3) “Which data would you love in a weekly vendor report?”

**Likely agent answers**

- 
    
    “Enquiries this week vs last, total views, viewings booked & attended.”
    
- 
    
    “Viewing feedback (liked/didn’t like), % serious buyers.”
    
- 
    
    “Offers received + buyer status (AIP/chain).”
    
- 
    
    “Recommended actions (price tweak, new photos, feature listing).”
    
- 
    
    “Sales progression status if under offer.”
    

**Yalla.House response**

- 
    
    **Report template fields (default):**
    
    - 
        
        Metrics: enquiries, viewings (booked/attended), average response time.
        
    - 
        
        **Feedback summary** + top objections.
        
    - 
        
        **Offers table** with buyer proceedability.
        
    - 
        
        **Market comps snapshot** (simple: 3 nearest actives/solds).
        
    - 
        
        **Agent recommendation** (free text) + “Approve change” button.
        
- 
    
    **Delivery:** automated every Friday 16:00, branded, CC negotiator.
    

# 4) “Where do leads fall through the cracks?”

**Likely agent answers**

- 
    
    “Branch mailbox gets flooded; after-hours leads aren’t picked up.”
    
- 
    
    “WhatsApp convos never get into the CRM.”
    
- 
    
    “Facebook/Instagram leads sit in DMs.”
    
- 
    
    “No one follows up the second time if the first call is missed.”
    

**Yalla.House response**

- 
    
    **Email-to-Lead parser** for portal & website emails.
    
- 
    
    **After-hours auto-reply** with pre-qual link & booking options.
    
- 
    
    **Light WhatsApp bridge** (for MVP: email-forwarding of transcripts or paste-in box on the lead).
    
- 
    
    **SLA rules** (e.g., 2-hour first response; breach → notify manager).
    
- 
    
    **Follow-up sequences** (D+1, D+3) with canned replies.
